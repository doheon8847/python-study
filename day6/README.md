## 모듈(module)
- 예를 들어 csv.py 파일이 있다면 이를 csv모듈이라고 부름
- 모듈을 사용하는 이유
    - 이미 잘 만들어진 소스코드를 재사용하기 위함이다
    - 비슷한 역할을 가지는 클래스나 함수를 하나로 모아서 관리와 사용을 편하게함



### 문법
'''

1 import [사용할 파일명]
 
2 약어를 쓰면
import [사용할 파일명] as [약어]

3 해당 파일의 특정 함수만 가져오기
from [사용할 파일명] import [함수이름]

4 해당 파일의 함수 전부 가져오기
from [사용할 파일명] import +
'''
